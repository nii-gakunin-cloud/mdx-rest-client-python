# mdx REST Client for Python

## examples  

mdx REST Client for Python 利用例を記載したノートブックファイルを同梱しています。（`examples`ディレクトリ以下）

- [commands.ipynb](./examples/commands.ipynb) ... コマンド一覧
- [deploy_vm.ipynb](./examples/deploy_vm.ipynb) ... 仮想マシンのデプロイ実行例
- [deploy_vm_for_vcnode.ipynb](./examples/deploy_vm_for_vcnode.ipynb) ... 仮想マシンのデプロイ＆VCノード（["`Node`"](https://nii-gakunin-cloud.github.io/ocs-docs/usermanual/#vcp-sdk-%E3%81%8C%E6%89%B1%E3%81%86%E6%A7%8B%E6%88%90%E8%A6%81%E7%B4%A0)）として利用するためのセットアップ例

## Installation

Pythonがインストール済みであることを前提としています。

### pip install  

pip installが可能です。

```
pip install git+https://github.com/nii-gakunin-cloud/mdx-rest-client-python.git
```
